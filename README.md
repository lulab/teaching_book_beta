### 预修课程准备

1. 基本生物课程：如《遗传学》或《分子生物学》

2. 基本统计课程：如《概率论》或者《生物统计》

3. 基本数学课程：如《线性代数》

4. 基本计算机课程：如《C语言》, Linux基础 等
